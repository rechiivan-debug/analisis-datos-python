#  Análisis de Datos con Python

Proyectos de análisis de datos usando Python, Pandas y SQL Server.

## Tecnologías
- Python 3.12
- Pandas
- SQLAlchemy
- SQL Server
- Requests (APIs)

##  Proyectos

### 1. Clima de Ciudades en Tiempo Real
**Archivo:** `prueba.py`
- Consumo de API pública (Open-Meteo)
- Extracción de datos en tiempo real
- Procesamiento con Pandas
- Exportación a CSV

### 2. Análisis de Ventas Northwind
**Archivo:** `analisis_ventas.py`
- Conexión a SQL Server con SQLAlchemy
- Merge de 3 tablas (Customers, Orders, Order Details)
- Análisis de ventas por país
- Pipeline ETL completo
